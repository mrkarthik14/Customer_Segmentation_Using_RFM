# Customer Segmentation using RFM

This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis.

## Structure
- `data/`: Contains raw data files.
- `notebooks/`: Jupyter notebooks for analysis.
- `sql queries/`: SQL scripts for data processing.
- `outputs/`: Generated results.
- `import_data.py`: Script to import data.
- `config.py`: Configuration settings.

## Setup
1. Install dependencies.
2. Run `import_data.py` to load data into the database.
